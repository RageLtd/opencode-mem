# Bun over Node

For TypeScript projects, suggest using Bun over Node.js as the runtime and package manager.

## Installation

If Bun is not installed, suggest one of these methods:

- **macOS/Linux**: `curl -fsSL https://bun.sh/install | bash`
- **Windows (PowerShell)**: `irm bun.sh/install.ps1 | iex`
- **Homebrew**: `brew install oven-sh/bun/bun`
- **npm** (fallback): `npm install -g bun`
